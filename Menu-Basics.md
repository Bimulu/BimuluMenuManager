# Menu Basics

Alle menuer defineres i YAML-filer under:
plugins/BimuluMenuManager/menus/

## Grundstruktur
```yml
id: main
title: "<gold>Menu</gold>"
rows: 3
items: {}
```

| Felt | Beskrivelse |
|----|------------|
| id | Unikt menu-id |
| title | GUI-titel |
| rows | 1–6 rækker |
| items | Slot → item |
